GUI Programming with Python: QT Edition


Now Shipping
The main topic of this book is application development using PyQt, a library extension to the Python programming language ? a library that is meant to form the basis for GUI programming. PyQt is free software, but there is also a commercial IDE available, BlackAdder, that is specially written to assist working with PyQt. I will show you the ins and outs of PyQt by developing a complete and complex application.

The combination of Python and Qt is extremely powerful, and is used in a wide variety of applications. People are scripting OpenGL applications with it, creating complex 3D models, animation applications, writing database applications, games, utilities and hardware monitoring applications. It is used in open source projects, but also by large companies, like Disney Television and Media. If you're not working on embedded software, hardware drivers or a new operating system, chances are that PyQt is the right choice for you, too.
