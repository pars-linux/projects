Conclusion

In this chapter we've taken a first sip of developing with Python and PyQt, touching lightly on many subjects. In the rest of this part, we'll investigate these issues in depth. Then we will start building real software in Part III. If you get lost, you can always refer to the online documentation that comes with Qt or BlackAdder. Appendix A, Reading the Qt Documentation, tells you how you can read the C++ class documentation with profit and pleasure.
