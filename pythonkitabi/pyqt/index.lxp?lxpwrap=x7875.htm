Conclusion

In this chapter we briefly investigated two ways of creating two-dimensional graphics with Python and Qt. If you possess a suitable 3d accelerated graphics card, you can also use the OpenGL Qt extensions from Python and create complex three-dimensional scenes with PyQt. I can't help you with that task — I've never worked with three-dimensional graphics myself, and besides, my computer isn't up to it.
