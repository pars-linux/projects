Name

%VersionCode -- Start of a version code _block_

Synopsis

%VersionCode

Description

The %VersionCode goes into sip_helper.cpp, which will be compiled into sip_helper.exe, which will produce sip$(module)Version.h, which will be included in (almost) all files. It should contain #defines corresponding to each of the different versions.
